So funktioniert das Orakel: Für jedes Produkt wird der Median-Preis berechnet. Jeder Preispunkt wird als Abweichung von diesem Median erfasst. Der Monatsindex zeigt den Median dieser Abweichungen pro Monat — über alle Produkte einer Kategorie. Index 100 = Jahresschnitt, 95 = 5% günstiger, 105 = 5% teurer. Datenquellen: Amazon-Preishistorie via Keepa (teilweise bis 2011).
